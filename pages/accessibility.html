Public Accessibility

Accessibility Statement

Website Accessibility

Walton County is committed to providing access to online budget information for all residents, including people with disabilities.

This budget website is designed to support accessibility and usability consistent with Web Content Accessibility Guidelines (WCAG) 2.1 Level AA. The County works to make budget information, tables, charts, links, documents, and interactive features usable by people who rely on assistive technology, keyboard navigation, screen readers, screen magnification, voice recognition software, or other accessibility tools.

Accessibility Design Guidelines

This website has been designed with accessibility in mind, including:

  • Keyboard-accessible navigation
  • Visible focus indicators
  • Skip-to-main-content links
  • Proper heading structure
  • Descriptive links and buttons
  • Accessible table headers and captions
  • Text alternatives for meaningful images

Browser Accessibility Information

Many popular browsers include built-in accessibility tools and keyboard settings. Users may review accessibility information for their preferred browser:

  • Chrome Accessibility Information
  • Firefox Accessibility Information
  • Microsoft Edge Accessibility Information
  • Safari Accessibility Information

Documents and Alternate Formats

Some materials linked from this website may be provided as downloadable documents. If you experience difficulty accessing a document, table, chart, or other material, you may request the information in an alternate format using the contact information above.

Supported Assistive Technology

This website is intended to support commonly used assistive technologies, including current versions of:

  • JAWS for Windows
  • NVDA for Windows
  • VoiceOver for macOS and iOS
  • Browser zoom and screen magnification tools
  • Keyboard-only navigation

Ongoing Accessibility Work

Accessibility is an ongoing effort. Walton County reviews and updates this website to improve access to public budget information. If you encounter an accessibility barrier, please contact us so the issue can be reviewed.
